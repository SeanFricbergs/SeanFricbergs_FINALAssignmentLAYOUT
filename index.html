Faster, more efficient building.

Layout has been designed to make web designers lives easier. Gone are the days of carefully trying to get a box the right size, place a div in a certain spot, or place several objects side by side. With Layout, it's as simple as clicking and dragging the object that you want to place and the code is generated for you.

Bring your dream site design to life.

With simple, easy to use tools, you can create any design or layout that your heart desires. Create simple or complex designs with familiar tools like resize, scale, and skew. The generated code is simple and organized, so even the novice web designer can add and alter it to their liking.

About the team.

A team of highly skilled web and graphic designers lead by Sean Fricbergs, a Humber College student, has worked long and hard to build this program.
